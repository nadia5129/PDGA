A web application built to display Professional Disc Golf Association (PDGA) event data in a clean, interactive format. 
The site organizes competitive disc golf history — showing past winners, event stats, purse data, divisions, and series pages — making it easier for users to explore tournaments.

This project was developed as part of my Software Development Capstone.

Features:

- Landing page for navigating event pages
- Series page displaying historical USDGC events
- Events, Champions, and Division tables built using PHP and MySQL
- Dynamic result data pulled from CSV and database queries
- Custom CSS styling for layout, UI components, and styles
- Search functionality for accessing series/events
- Organized file structure for scalability

Tech Stack:
- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Data: CSV files
- Tools: GitHub, cPanel, phpMyAdmin

Project Structure:
- CSS/        # Styling & page layout
- JS/         # Front-end scripts
- USDGC/      # Landing page & series pages
- data/       # CSV datasets for events/results

How to run:
- Clone or download repository
- git clone https://github.com/nadia5129/PDGA.git
- Place folder inside server environment (XAMPP/MAMP/cPanel/public_html)
- Ensure CSV files remain inside /data
- Open in browser at:
http://localhost/PDGA/USDGC/landing.php
